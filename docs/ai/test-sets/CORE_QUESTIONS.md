# Test Set Base - Preguntas Canónicas

> **Actualizado:** 2026-02-18

Usar este set para validar cada release del asistente IA.

## CRM

- Como paso un prospecto a cliente?
- Donde creo un cliente nuevo?
- Que pasa despues de crear una cuenta?
- Como creo un lead?
- Donde veo los negocios (deals)?

## Ops

- Donde veo las pautas?
- Como funciona el sistema de turnos?
- Donde reviso asistencia diaria?
- Como asigno un guardia a un puesto?
- Que es PPC?

## Marcacion

- Como marca un guardia su entrada?
- Donde configuro el PIN de un guardia?
- Como genero el QR de una instalacion?
- Que pasa si el guardia no tiene GPS?

## Rondas

- Como armo checkpoints de rondas?
- Donde programo una ronda?
- Como veo si se cumplio una ronda?
- Como armo una plantilla de ronda?
- Donde veo alertas de rondas?

## Tickets

- Como creo un ticket?
- Que es el SLA de un ticket?
- Donde veo tickets pendientes de aprobacion?
- Un guardia puede crear tickets?

## Notificaciones

- Como configuro mis notificaciones?
- Que tipos de notificaciones existen?
- Puedo desactivar las notificaciones por email?

## Finanzas

- Que rendiciones faltan por aprobar?
- Donde apruebo rendiciones?
- Que estado tiene una rendicion?
- Como exporto pagos al banco?

## Personas

- Donde doy de alta un guardia?
- Que estados puede tener un guardia?
- Como funciona la lista negra?
- Donde veo los documentos de un guardia?

## Datos duros

- Cuantos guardias activos hay?
- Cual es la UF y UTM de hoy?
- Cuantos tickets abiertos tenemos?

## Criterio de aprobacion

- Debe responder en espanol claro.
- No debe inventar datos.
- Debe usar links clickeables cuando aplique.
- Debe dar pasos accionables en preguntas de "como".
- Debe cubrir los 14 módulos implementados.
